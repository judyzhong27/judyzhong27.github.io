Word Hippo Redesign

UI Design Inspiration of the Week

Word Hippo is an online service I use regularly to generate synonyms and antonyms for things like cover letters, newspaper articles, and school reports. I've been using the site for a few years now and I thought the UI could use a little change up so I did this redesign as my inspiration for this week. The gradient background was created on Adobe Illustrator and the site was made in Sketch.

Tools Used in this Project

Figma

I used Figma the web app and Figma Mirror to create an iPhone X prototype of Snack Overflow. Initially I used Sketch for the iOS UI Design elements but decided to import into Figma to implement the connections I wanted between screens in the prototype, which can be seen through this link

Sketch

With the Sketch app, I created promotional materials by importing the screens made from Figma into iPhone X mockup sets. All the materials on this page with Snack Overflow content was developed in Sketch

Adobe Illustrator

The Snack Overflow logo and all the icons within the app were self-created on Illustrator, with the exception of the menu and arrow icons, which were taken from Google's material design icon pack found here
